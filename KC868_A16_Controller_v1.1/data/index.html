Configure Input Interrupt

Priority Levels:

  • High: Processed first when multiple inputs change
  • Medium: Processed after all high priority inputs
  • Low: Processed after all high and medium priority inputs
  • None: No interrupt, input is polled periodically

Trigger Types:

  • Rising Edge: Triggers when input changes from LOW to HIGH
  • Falling Edge: Triggers when input changes from HIGH to LOW
  • Change: Triggers on any state change (both edges)
  • High Level: Triggers continuously while input is HIGH
  • Low Level: Triggers continuously while input is LOW

Configure HT Sensors

Sensor Types:

  • Digital Input: Use as a standard digital input
  • DHT11: Basic temperature/humidity sensor (±2°C, ±5%RH)
  • DHT22: Higher precision temperature/humidity sensor (±0.5°C, ±2%RH)
  • DS18B20: Precision temperature sensor (±0.5°C)

Note: Changing sensor type will reset any associated schedules or triggers.
